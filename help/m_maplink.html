Plotting Maps in ZMAP using the M_MAP software

ZMAP maps are mostly plain xy plots, no map projection re available. The MATLAB mapping toolbox addresses this need, but is rather expensive and a bit cumbersome. GMT is an elegant way to plot nice looking maps, but requires to export all data from ZMAP. The M_MAP software is a nice set of tools that produce nice looking maps with little effort.

In a similar fashion, color maps can be plotted using the m_map features. To change the projection used, modify the file /m_map/ploymymap.m. See the M_Map documentation for available projections.

To plot these maps with pathced landed area, the low and intermediate resolution GHSSS database of coastline data needs to be within the Matlab search path, for example in the dem directory.
