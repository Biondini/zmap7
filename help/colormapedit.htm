Editing Colormaps

ZMAP makes use of the makecmap function written by Colin Humphries. To modify a colormap, select the 'Figure' -> 'Edit Colormap' option from the Menu Bar. This will open the ColorMap Editor shown below. Any change in this interface will immediately be applied to the colormap of calling figure. To change the colormap under investigation, you can move an existing handle (white circles) using the left mouse button, add a new break point (middle button), or delete a handle (right button - PC and MAC - I am not sure what to do...).

More functions are available from the command line prompt - type help makecmap for additional info.
