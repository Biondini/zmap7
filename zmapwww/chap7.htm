Chapter 6 Declustering an Earthquake Catalog

 I

Introduction

To separate the dependent from the independent seismicity, the earthquake catalog can be declustered. The decluster algorithm used in ZMAP is based on an algorithm developed by Reasenberg (Reasenberg, 1985). Most of the software for the declustering has been developed by Alexander Allmann.

When you choose "Decluster the catalog" from the "Tools" pulldown menu of either the map or cumulative number windows, an input parameter box will appear. Eight different parameters can be set:

    • taumin: the look-ahead time for un-clustered events in days (default is one day).
    • taumax: the maximum look-ahead time for clustered events in days (default 10 days)
    • P1: a measure of the confidence that you are observing the next event in the sequence (default is 0.95)
    • xk: the factor used in xmeff (default 0.5)
    • xmeff: the "effective" lower magnitude cutoff for the catalog.
    • rfact: the factor for the interaction radius of dependent events (default is 10)
    • Epicenter-Error: horizontal location error in km (default is 1.5)
    • Depth-Error: vertical error in hypocenter location in km (default is 2) For additional information about these parameters hit "Info".

      Press "Go" to start the declustering. Depending on the speed of your computer and the size of the catalog, this may take some time. A wait-bar will indicate what percentage of the declustering is completed.

Tools from the Cluster Window

The options "Select" and "Cuts" in this window are based on the equivalent events of the clusters. The magnitude of an equivalent event is based on the cumulative seismic moment of all earthquakes in the cluster. The center of the sequence is the location of the equivalent event.

Symbol: Changes the symbol size and type.

Select: Lets you select a subset of earthquakes using polygons or circles. See the description about how to select earthquakes in chapter 4. All clusters will be selected whose equivalent event is in the marked area

Histogram: Displays histograms of the magnitudes of the biggest events, the equivalent event magnitudes, depth, duration, foreshock duration and foreshock percentage

Tools: Plots the cumulative number curve and allows a b-value estimation. See the "Cumulative Number Window" section for details.

Display: Allows you to zoom into the map using a rubberband zoom, refresh the plot, and show and hide the map window. Back will reset the catalog to the initial selection.

Cuts: You can create a subset of all the clusters. You can cut in time, magnitude, depth, or number of earthquakes related with the cluster.

Special: You can select a subset of all the clusters. You can display only
regular sequences (foreshocks, aftershocks, whole sequences), doublets, or swarms. Using "Single" allows to select a single cluster of earthquakes either with the mouse or by entering the cluster number, and provides a new environment for some further investigations of the selected cluster

Seismicity Map (Cluster)

This window displays a seismicity map either of the original or the declustered catalog. In addition to most of the routines already known from the regular seismicity map it gives you the opportunity of a selection of sub-catalogs with parameters corresponding to the present cluster window. It is therefore easy to inspect the quality of the declustering in small areas of interest or to compare cluster features with the respective features in the background seismicity.

Select: Area selection with polygons or circles as described in chapter 4. "Cluster Window Values" selects a sub-catalog with the same time, area and magnitude limits used in the current cluster window. "Expanded Cluster Window Values" expands these limits by 0.2 degrees, 0.2 years, 0.2 magnitudes and 10 km in depth. With "Select new parameters" you can change all parameters of the current data set. The default values are the current limits. All these options work with the original or the declustered catalog.

Catalog: "Reset Catalog" sets the original catalog and "Declustered Catalog" the declustered catalog to the current one. This selection has to be made first, otherwise any following selection will be undone.

Tools: all options here are described in chapter 4.

.Single Cluster Examination

This window allows a detailed examination of a specific cluster

FIGURE 48. Single cluster examination window

Aftershocks are displayed in red, foreshocks in blue. The pink cross is the location of the largest earthquake in the selected sequence. The "After" and "Fore" buttons allow to work only with the respective events. This selection must be done before area selections and cuts, which can be made at any time.
Options: You have the opportunity of area selections, cuts, histograms, cumulative number plots, b-value and p-value estimation. Different from the "Cumulative Number Window", the option "p-value automatic" requires no further input and displays the result in the familiar way.

Hit "Back" to go to the original Cluster Window and hit "Close" to leave the single cluster examination tool.
