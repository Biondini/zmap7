Chapter 4 The Map Window

 The map window (Figure 20) displays the seismicity in the selected catalog, and plots an overlay of faults, coastlines, rupture symbols etc. This overlay can be customized by changing the file 'src/overlay_.m'. The map window is used to interactively design the size of a grid, and select earthquakes. Several pull-down menus are available from here. Figure 19 shows an example of how to change the symbol type of the earthquakes plotted in the map window. Many of the options are self-explanatory and will not be discussed in further detail. The options under "Tools" will be mostly discussed in separate chapters.

FIGURE 19. Pulldown menu to change the symbol type used to plot earthquakes in the map window

FIGURE 20. The map window

Layout Tools

The gray shaded menus are created using the package `eztools' available from the MathWorks anonymous ftp site. In order to create this menu bar in a figure, you only need to type `matdraw' in the matlab command window. These functions make it possible to add or modify text as well as change the general appearance of a figure before printing. Most of the functions are pretty much self-explanatory, and here only a few highlights will be pointed out.

Changing Symbol Sizes

The size of the symbols used to display earthquakes can be changed using the symbol menu. In addition, you can change the type and color of the symbols used, and change the type of the legend used (time, depth, or magnitude). The `Change Background colors' option makes it possible to change the background color of the windows (default: gray 0.9 0.9 0.9) and the axes (default: light yellow, 1 1 0.6).

Selecting Earthquakes

The map window allows you to select earthquakes in space using four methods:

Select earthquakes in circle (fixed ni): Selects the ni closest earthquakes to a point that you choose in the map. "Circle" is a fast and easy tool to sample the seismicity rate in various areas. Hit "Circle", wait until the cursor changes into a cross, and click where you would like the circle centered. The 100 closest epicenters in space over the entire length of the catalog are marked by black symbols in the map and the cumulative number curve is displayed (Figure 34). You can vary ni by editing the number displayed in the upper right corner of the map window and hitting "set ni".

Select earthquakes in circle (Menu): This will open a menu where you can either select the ni closest earthquakes to a center point, or select earthquakes within a fixed radius from the center-point. Coordinates of the center can be entered using the mouse or using the keyboard.

FIGURE 22. Circle input menu

Select earthquakes in polygon: Allows you to select earthquakes within a polygon of any shape. After clicking on "Select", the cursor will change into a cross. Select now the corners of your polygon using the left mouse button. Use the right mouse button to select the last point of the polygon (Macintosh users: select any key on the keyboard to select the last point). The polygon outline is displayed and the events inside are highlighted in the map. In addition, the cumulative number curve is displayed in the cumulative number window (Figure 34).

Select earthquakes in polygon (Menu): Allows you to input the coordinates of a polygon interactively by mouse, enter them via keyboard, or load a file with the coordinates. If you decide to import the coordinates of the polygon with the keyboard, press "More" to add another corner to the polygon, press "Last" after the last entry.

FIGURE 23. Polygon input menu

The Catalog Menu

Refresh: Refreshes the window display

Reset catalog: This option will reset the catalog to the initial selection made in the general parameter window. Use this button to reset the catalog after working with a subset of data in space, time, depth or magnitude.

Open new catalog: Opens a load datafile window to select a new catalog.

Select new parameters: Resets the catalog to the original loaded catalog prior to the selection in the general parameter window. You can apply a new selection of general parameters.

Save selected catalog (*.mat): Opens a save data window to input a filename for the catalog to be saved. The saved catalog will contain the current selection of earthquakes as well as fault-lines, coastlines, and main fault symbols.

Save selected catalog (ascii): Opens a save data window to input a filename for the catalog to be saved. The current catalog only will be save in ascii format.

The Tools - an overview

A number of tools can be accessed from the map window. Most of them are described in more detail in subsequent chapters of this manual.

Plot cumulative number: Plots the cumulative number of the current catalog. See "The Cumulative Number Window" for details. Run GENAS: See the GENAS section for details. Monte Carlo: Compares the distribution of z - and b-values with synthetic catalogs. See the chapter on synthetic catalogs for details.

Create Cross-Section: This will create a cross-section using a Lambert projection. You can use the projected events to perform a b- or z-value analysis in a cross-section view. 3D View: Displays the earthquakes in a 3-dimensional view. Time Series: Creates Time-Depth, Time-Magnitude, and Mean-Depth plots. Summary Plot: Plots a number of seismicity parameters in one summary figure. Mapping z-values: This submenu is used to create a z-value map in either map-view or cross-section. Datafiles containing cumulative number grids created in earlier session can be loaded. Mapping b-values: Used to investigate the frequency-magnitude distribution using grids in map view or cross-section. Existing grids can be reloaded. Decluster the Catalog: Opens a decluster input parameter window. See the chapter on Declustering for details. Misfit calculation: This function will calculate the misfit of each individual event to a assumed homogeneous stress tensor. See the chapter on stress tensor inversions and the cumulative misfit method for details. get coordinates with cursor: The coordinates of the point where you click with the mouse will be displayed in the Matlab command-line window. Zmapmenu: Shows the menu of different z-value maps. You need to create a grid first to use this option. SEISMOLAP: Initiate investigation of seismicity rate changes using the SeismoLap function (Zschau, 1995), either at one point or for a grid. See the chapter on SeismoLap for details. Histogram: You can plot histograms of the magnitude, depth, and time distribution of the current catalog. FIGURE 24. Histogram of the magnitude distribution

Below is a more detailed description of some of the tools.

Mean Depth Investigation

Mean depth can be investigated as a function of time. A sliding window containing a fixed number of earthquakes will be moved in steps of n earthquakes over the entire time-series. When you choose "Mean Depth" from the "Tools" pull-down menu of the map window, an input parameter box will allow you to select the number of earthquakes in each averaging window, as well as the step in days.

FIGURE 25. Mean depth input parameter box

Hit "Go" to start the analysis using the current selection. A waitbar will indicate the progress of the mean depth calculation. When the calculation is completed, a new window will display two plots: the depth of the earthquakes as a function of time, and the mean depth and standard deviation as a function of time.

FIGURE 26. Mean depth output window

Selecting The 'comp' option in Figure 26 will compare the depth distribution of two given periods. The two depth histograms for both periods are displayed (Figure 27) as well as the difference between the two histograms. This makes it possible to detect shifts in the depth distribution of earthquakes.

FIGURE 27. Comparison of the distribution of earthquakes with depth for two periods. The bottom histogram is the difference between the two above.

3D View

The 3D view option displays a 3 dimensional view of the currently selected seismicity (Figure 28). You can change the angle of view using the viewer window. The color code of the symbols will be according to the depth or time of the events, depending on the latest selection in the map window.

FIGURE 28. 3-D plot of the seismicity and viewer window to change the angle of view

Time Depth Plot

This option will display a time depth plot of the seismicity (Figure 29).

FIGURE 29. Time depth plot

Time Magnitude Plot

This option will display a stem plot of the magnitude distribution as a function of time and depth (Figure 30).

FIGURE 30. Time Magnitude Plot

Summary Plots

A comprehensive summary of the seismicity can be viewed by selecting the `Summary Plot' option from the Tools Menu. The seismicity will be shown in a variety of time-series, a map, and a b-value plot. The objective of this summary is to obtain a quick overview over the changes in the seismicity, important for example for monitoring the seismicity during a volcanic crisis.

FIGURE 31. Summary plot of the seismicity at Iliamna volcano during the 1996 earthquake swarm.

Cross-Sections

From the Tools Menus of the Map window the option "Create Cross-Section" can be selected. A new window will display the seismicity using a Lambert projection. You can either choose the `Select endpoints by mouse' or "coordibardinate input" option by clicking on the menu ("select an option") displayed in the lower right corner of the figure. The two endpoints (labled A and B) of the cross-section need to be defined with a mouse-click. The width of the cross-section is the total width of the box. It can be redefined in the field in the lower right corner. The selected events are displayed on the map as red crosses and the cross-section window will pop up (Figure 33). A new selection can be initiated by hitting the 'New selection' button in the Seismicity Map (Figure 20). The 'Cross-section Window' is the starting point for b- and z-value analysis using the cross-section (See "b-values in cross-section view" on page 85.).
